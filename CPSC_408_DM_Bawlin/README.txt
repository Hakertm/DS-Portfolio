!!!WARNING!!!
There is practically no error checking in this app. Some elements will
still work even if you mess up, like searching for a nonexistent record
or not filling out every entry box. However, some elements, like inputing 
a non-integer into an integer entry will be problematic. The app won't 
crash or break perse, but it is something to be aware of. So if you want
the best experience, simply use common sense.
